Overview:-
Analyzed FIFA World Cup data (1930–2024) using Tableau to gain insights into tournament history, team performance, and audience trends.

Tools:-
Tableau, MySQL

Key Features:-
Cleaned and processed raw data using MySQL
Explored host countries, tournament years, and match outcomes
Analyzed team performance: applications, qualifications, finals, and semi-finals
Examined World Cup winners, stadium details, and final match opponents
Evaluated total matches played and goals scored
Created interactive dashboards and visualizations to summarize insights
